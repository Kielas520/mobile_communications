# 最大多普勒频移公式的来历

> 来源：第2章小结第6题"GSM 载频 900MHz、车速 100km/h，求最大多普勒频移"。
> 参考转录：`source/extracted/移动通信复习-马志乐/手写笔记转录.md`（多普勒频移公式 + 考点第6题）。

## 一、什么是多普勒频移（先理解现象）

移动台在动，它和波源之间的距离一直在变化，导致接收到的信号频率偏离了原始载频。
生活类比：火车鸣笛，驶近时听到的声音变尖（频率升高），驶离时变闷（频率降低）——这就是多普勒效应。无线电波同理，这个频率偏移量叫**多普勒频移** $f_d$。

## 二、公式是怎么推出来的

设移动台以速度 $v$ 运动，一束电波从与运动方向成夹角 $\alpha$ 的方向入射。

**第1步：算位移**
在一小段时间 $\Delta t$ 内，移动台沿运动方向走的距离：
$$d = v\,\Delta t$$

**第2步：算传播路程的变化（取投影）**
信号传播路程的变化量，是这段位移在"波来向"上的投影：
$$\Delta l = d\cos\alpha = v\,\Delta t\cos\alpha$$

**第3步：路程变化 → 相位变化**
电波每走一个波长 $\lambda$，相位变化 $2\pi$。所以路程变 $\Delta l$ 对应的相位变化：
$$\Delta\varphi = \frac{2\pi}{\lambda}\,\Delta l = \frac{2\pi v\,\Delta t\cos\alpha}{\lambda}$$

**第4步：相位变化率 → 频率偏移**
频率等于相位变化率再除以 $2\pi$：
$$f_d = \frac{1}{2\pi}\cdot\frac{\Delta\varphi}{\Delta t} = \frac{v}{\lambda}\cos\alpha$$

这就是多普勒频移的基本公式。

## 三、最大多普勒频移

当移动台正对着波来方向运动（$\alpha = 0$，$\cos\alpha = 1$）时，频移最大：

$$f_m = \frac{v}{\lambda}$$

再用 $\lambda = \dfrac{c}{f_c}$ 代换，可写成只含速度和载频的形式：

$$\boxed{\,f_m = \frac{v}{\lambda} = \frac{v\,f_c}{c}\,}$$

- $v$：移动台速度（m/s）
- $\lambda$：波长（m），$\lambda = c/f_c$
- $f_c$：载波频率（Hz）
- $c = 3\times 10^8\,\mathrm{m/s}$：光速

## 四、本题代入计算

已知 $f_c = 900\,\mathrm{MHz}$，$v = 100\,\mathrm{km/h}$。

**① 速度换算成 m/s**（除以 3.6）：
$$v = \frac{100}{3.6} = 27.78\,\mathrm{m/s}$$

**② 算波长**：
$$\lambda = \frac{c}{f_c} = \frac{3\times 10^8}{900\times 10^6} = \frac{1}{3}\,\mathrm{m} \approx 0.333\,\mathrm{m}$$

**③ 算最大多普勒频移**：
$$f_m = \frac{v}{\lambda} = \frac{27.78}{1/3} = 27.78\times 3 \approx 83.33\,\mathrm{Hz}$$

> 提醒：km/h 换 m/s 一律除以 3.6（因为 $1\,\mathrm{km/h} = 1000/3600\,\mathrm{m/s}$）。转录里指出手写稿此处速度换算疑似笔误，正确值为 27.78 m/s。

## 五、与本课程的关联

- 多普勒频移属于**小尺度衰落**的成因之一，反映移动带来的频率扩展。
- $f_m$ 越大，信道变化越快（相干时间越短），对系统的时变性、误码、均衡设计影响越大。
- 注意区分：$f_d = \dfrac{v}{\lambda}\cos\alpha$ 是某方向来波的频移（可正可负），$f_m = \dfrac{v}{\lambda}$ 是其最大值（恒正）。
